Introduction

Welcome to the Translation Tools Experiment! Thank you for volunteering to participate.

To take part in this experiment, please send an email to daniel.vidal.hussey at gmail dot com and you will be provided with a participation link. You can read the information below before starting.

We estimate the total experiment time to be between 40-50 minutes depending on translation speed. It is not possible to stop the experiment and continue later, please only start when you have enough free time to complete the whole experiment.

Below we provide a summary of the experiment steps.

Prodecure

  1. Initial questionnaire
  2. Description of translation brief
  3. Text 1 translation (EN -> ES)
  4. Text 2 translation (EN -> ES)
  5. Text 3 translation (EN -> ES)
  6. Final questionnaire

Notice

It is not possible to close the survey and finish it later. Please only start the experiment when you have time to complete it.
